什么事Prompt：

提示词工程



Function Call：

理解语义、自主决策、机构化调用工具、询问用户、补全信息、基于返回的结构化信息，整合生成回复、



Function Call的成功率非常依赖基座LLM

Function Call可以让LLM变成智能中枢





LLM Based Agent

从无状态到特定状态的定义和识别
