如何获取 Token：
  1. 打开 GitHub → Settings → Personal access tokens
  2. 勾选 repo 权限（完整仓库访问）
  3. 点击 Generate token → 复制 token
  4. 粘贴到上方输入框，点登录

⚠️ Token 仅存储在浏览器本地，不会上传到任何服务器
