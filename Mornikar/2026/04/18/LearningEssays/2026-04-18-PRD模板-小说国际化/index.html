[项目名称] PRD

1. 项目背景

  • 解决中国小说国际化
  • 目标用户是国外小说用户

2. 核心功能清单

功能 优先级 描述
小说板块 P0
推送书架 p0
书籍详情页
收藏书架 P0
长文本中文转英文功能 P0 可以让长小说由中文翻译成英文
阅读器 p0 阅读器可以读取市面上大部分书籍格式
悬浮AI P0 选中文字可以复制
后台板块 P0
小说发布功能 P1 用户可以整本或者章节上传发布，期间可以选择是否使用翻译功能

用户登录功能 p1 重要

开发者后台管理 p2

3. 功能详细描述

3.1 [小说板块]

  • 用户故事：普通读者用户通过游客或者账户登录进入首页，用户可以通过点击小说板块进入默认的小说板块，在推荐书架上通过类别过滤获取想要 书籍，点击进入书籍详情页
  • 输入：登录进入书籍推荐页面
  • 处理：小说板块的推荐系统，类别分类
  • 输出：进入书籍详情页，根据用户id定位上次阅读章节
  • 边界情况：推荐系统模块，类别分类充足

3.1 [推送书架]

  • 用户故事：普通读者用户通过游客或者账户登录进入首页，通过游客用户或者初次登录者进入的用户会选择偏好作为推送系统的偏好，推送书架会根据偏好书签进行推送
  • 输入：偏好标签
  • 处理：小说板块的推荐系统，类别分类
  • 输出：进入书籍详情页，
  • 边界情况：推荐系统模块，类别分类充足

3.1 [小说发布功能]

  • 用户故事：作为国内后台用户，我想将国内小说整本上传，可以在后台转化为英文版，或者直接整本上传英文版
  • 输入：用户上传小说文档，或者文字粘贴长文本上传
  • 处理：AI识别内容，按章节切段进行处理，翻译中文转化英文
  • 输出：翻译后进入编辑页面，编辑页面限制不允许复制内容，有下一步是发布
  • 边界情况：文本过大自动做切段处理，中文转英文翻译能力
